# Smart School Cafeteria Access System

Sistema de control de acceso al comedor escolar basado en códigos QR, diseñado para mejorar el flujo de estudiantes, evitar abusos del servicio y registrar datos de uso del comedor.

## Problema

Durante el horario de almuerzo se presentan varios problemas:

- Filas largas y desorganizadas
- Estudiantes repiten su porción
- Estudiantes sin beneficio acceden al comedor
- No existen registros confiables de uso

Este proyecto propone una solución automatizada basada en visión por computadora y verificación de QR.

## Características

- Identificación de estudiantes mediante QR
- Verificación automática de autorización
- Prevención de repetición de comidas
- Detección de accesos sin autorización
- Registro de accesos al comedor
- Exportación de datos para análisis futuro

## Arquitectura del sistema

El sistema está dividido en módulos:

- Módulo de lectura de QR
- Módulo de verificación
- Módulo de registro
- Módulo de seguridad
- Módulo de interfaz

## Objetivos del proyecto

- Automatizar el control de acceso al comedor
- Reducir el tiempo de espera
- Evitar abusos del servicio
- Generar datos para análisis futuro
